# Requirements
{: .no_toc }

---

Before installing Sol Cerberus make sure you meet the following requirements:
## Anchor

| Software       | Version |
|:---------------|:--------|
| [`Rust`] | `1.67.0` or later |
| [`Anchor CLI`] | `0.26.0` or later |
| [`Solana CLI`] | `1.14.13` or later |

## Frontend

| Software       | Version |
|:---------------|:--------|
| [`Node JS`] | `16.19.0` or later |
| [`Yarn`] | `1.22.17` or later |
